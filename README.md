# Weather App

Current conditions and 7-day forecast for any city, powered by [OpenWeatherMap](https://openweathermap.org/).

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Zustand

## Setup

```bash
pnpm install
```

Add your API key to `.env.local`:

```
OPENWEATHERMAP_API_KEY=your_key_here
```

Requires an [OpenWeatherMap One Call 3.0](https://openweathermap.org/api/one-call-3) subscription.

```bash
pnpm dev
```
